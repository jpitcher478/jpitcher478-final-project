# ICT-280-M70 - Final Project

## Overview
For my Final Project in this course, I created a multi-page website based on a client profile. The client profile specifies a website for a fictional company called GreenTech Solutions, whose mission is to help businesses reduce their environmental impact through innovative technology. The website is designed to provide information about GreenTech Solutions and its products and services while visually representing the company’s environmentally focused aesthetic.

## Features

- Multi-page structure
- Images
- Responsive mobile-friendly design
- Navigation menu across all pages
- List of products and services
- Filter functionality using JavaScript
- Company timeline table
- Educational resources
- Contact information page
- Accessible color choices

## Technologies Used

- HTML5
- CSS3
- JavaScript
